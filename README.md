# 🚚 Delivery App

A C# console/desktop application for managing deliveries across three user roles: **Customers**, **Clients**, and **Deliverers**.

## Features

- **User authentication** — login and account creation for all user types
- **Order management** — place, track, and manage orders and items
- **Multi-role support** — separate functionality for customers, clients, and deliverers
- **Ratings system** — customers can submit ratings for orders/deliveries
- **OOP design** — built around core object-oriented principles (encapsulation, inheritance, polymorphism)

## User Roles

| Role | Capabilities |
|---|---|
| **Customer** | Create account, login, place orders, submit ratings |
| **Client** | Manage items and incoming orders |
| **Deliverer** | View and fulfil assigned deliveries |

## Tech Stack

- **Language:** C#
- **Paradigm:** Object-Oriented Programming
